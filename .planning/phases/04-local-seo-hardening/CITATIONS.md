## 4. Consistency Rules

Follow these rules EVERY time you submit to a directory.

### Name Consistency
- Always use **"Breeze Roofing"** exactly
- DO NOT use variations like "Breeze Roofing NC", "Breeze Roofing LLC", or "Breeze Roofing, Inc."
- Legal name may differ in incorporation docs, but public-facing name must be consistent

### Address Consistency
- Always use **"1234 Market Street, Wilmington, NC 28401"** exactly
- DO NOT abbreviate "Street" to "St." or use P.O. boxes
- DO NOT use virtual office addresses or alternative addresses

### Phone Consistency
- Always use **"(910) 665-5277"** formatted exactly this way
- DO NOT use variations like "910-665-5277" or "9106655277" or "+1 (910) 665-5277"
- If a directory strips formatting, that's acceptable, but always enter formatted version

### Website Consistency
- Always use **"https://breezeroofingnc.com"** (include https://)
- DO NOT use "www." prefix unless required by directory

### Category Selection
- **Primary category:** Select "Roofing Contractor" or closest equivalent
- **Secondary categories:** Add "Home Improvement" and "Construction Company" when available
- Avoid unrelated categories (do not add categories just to appear in more searches)

### Photos and Logo
- Upload the SAME logo file to every platform (consistent branding)
- Use the SAME set of project photos across platforms when possible
- Minimum: Logo, 3-5 project photos, optional team photo

## 6. Tips for Submission

### Claim Existing Listings First
Some directories may already have a listing for Breeze Roofing (created automatically from public data sources). Always **claim existing listings** before creating new ones to avoid duplicates.

### Verification Methods
Most directories verify ownership via:
- **Phone verification:** Automated call or text with code
- **Email verification:** Link sent to business email
- **Postcard verification:** Physical postcard mailed to business address (Google uses this)

Allow 5-10 business days for postcard verification methods.

### Avoid Paid Directories Initially
Focus on free listings first (Tier 1 and most of Tier 3). Evaluate paid directories (Angi, HomeAdvisor, BBB) after confirming they drive actual leads in your market.

### Documentation
Save screenshots of completed listings and verification emails. This creates a record in case of disputes or if you need to prove ownership later.

## 7. Common Issues and Solutions

**Issue:** Directory already has a listing, but I can't claim it
**Solution:** Look for "Claim this business" or "Is this your business?" link. Contact directory support if link is missing.

**Issue:** Directory requires different address format than our standard
**Solution:** Enter as close to standard as possible. The important part is street number, street name, city, state, and zip are correct.

**Issue:** Can't find exact category "Roofing Contractor"
**Solution:** Use closest equivalent: "Contractor", "Roofing Service", "Home Improvement", or "Construction"

**Issue:** Directory rejects phone number format
**Solution:** Try entering without formatting: 9106655277. Most directories auto-format for display.

**Issue:** Duplicate listings exist
**Solution:** Claim all duplicates if possible, then request merge or removal of extras through directory support.
